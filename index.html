Hi there! I'm Stephen, a front-end developer and obsessive hiker. Coming from the museum and art gallery world, I have an eye for clean design and knack for producing meaningful experiences. Instead of measuring tape and drywall, I now employ semantic html5 and easy to follow css + sass to sculpt user-friendly digital spaces. I'm driven to problem solve and help others to bring complex technical visions to life with React JS and modern ES6 Javascript in teams using version control best practices with git + github
