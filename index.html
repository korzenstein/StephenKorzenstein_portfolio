Hi there! I'm Stephen - a front-end developer and avid hiker. Coming from the museum + art gallery world, I have an eye for clean design and a knack for producing meaningful experiences. Instead of measuring tape and drywall, I now employ semantic html and easy to follow css + sass to sculpt user-friendly digital spaces. Collaborating in teams, my process is guided by command line, clear comminication and github best practices. I'm driven toproblem solve thorny questions and find joy in helping others bring complex technical visions to life with React and vanilla Javascript
